# Applied Intelligence Website

This repository contains the public-facing website for the Applied Intelligence ecosystem.

## Role

The website serves as the framework shell and external presentation layer for Applied Intelligence.

It is not the application runtime and does not contain the execution spine.

## Related Repositories

- applied-intelligence-app → application runtime, AI-Connect module, execution spine
- kool1160-applied-intelligence → master source hub, shared bundles, ecosystem definitions
- applied-intelligence-website → public website and visual shell
