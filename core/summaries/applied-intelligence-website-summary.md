# Applied Intelligence Website Summary

The Applied Intelligence website serves as the public-facing framework shell for the ecosystem.

## Purpose
- Present the ecosystem clearly
- Show agent roles
- Communicate system value

## Key Rules
- Do not contain execution logic
- Do not duplicate app runtime
- Reflect source JSON accurately
- Preserve `.html` navigation and real page content
- Do not add binary assets

## Visual Direction
- Clean, minimal layout
- Apple-inspired spacing
- Dark navy / glass UI
- Homepage should read closer to a dashboard system shell than a generic marketing landing page
- Strengthen top glass navigation, bottom dock, system overview sections, and agent card grid hierarchy

## Homepage Messaging Guardrails
- Primary headline: `Standardize to Optimize.`
- Supporting positioning (not headline): `Built from the floor up.`
- Product capitalization: `AI-Connect` and `AI-Trace`
- Contact email should be omitted unless an approved domain/email is explicitly provided
